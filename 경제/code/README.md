# 코드 설명

## exchange_rate_basic.ipynb
USD/KRW 환율 과 달러 인덱스의 비교 / 통계 분석
외환보유고량 비교

## foreign_exchange_voalatility.ipynb
환율의 변동성 분석

## fxfi_basic_analysis.ipynb
환율 공포지수 설계 및 lead-lag 상관계수 분석

## fxfi_volatility.ipynb
환율 공포지수 변동성 분석

## fxfi_correlation.ipynb
환율 공포지수와 실제 환율의 cross correlationship 분석 및 rolling z-score 분석

## granger-casualty,ipynb
환율 공포지수와 실제 환율 중 어떤 것이 선행하는지 파악하기 위한 그래인저 인과관계 test

## residual_macro_analysis.ipynb
환율 공포지수로 설명이 안되는 환율의 움직을 설명하기 위한 거시 경제 지표 분석

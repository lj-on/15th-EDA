구조, 의미, 그리고 맥락: 가사로 조명한 한국 대중가요 30년의 지형도

프로젝트 개요 본 프로젝트는 1996년부터 2025년까지의 멜론 차트 데이터를 기반으로 한국 대중가요 가사의 구조적·의미적 변화를 분석하여, 최근의 이지 리스닝 트렌드와 영어 가사 혼용 등의 현상을 데이터로 입증하고 시각화하는 것을 목표로 함

주제 선정 배경

1.1 연구 배경

K-POP의 위상 변화: 글로벌 시장에서의 성공과 달리, 국내에서는 가사의 의미가 모호해졌다, 영어가 너무 많다는 비판이 제기됨

이지 리스닝 트렌드: 짧은 러닝타임과 단순한 멜로디, 반복적인 가사가 주류를 이루면서 가사의 서사성이 약화됨

1.2 연구 필요성

대중가요 가사의 변화를 30년 시계열 데이터로 정량적으로 입증함

가사의 구조적 복잡도와 어휘 다양성을 통해 시대별 작사 스타일의 진화 과정을 추적함

데이터 개요 및 전처리

데이터 출처: 멜론 월간 차트 TOP 100

수집 기간: 1996년 1월 ~ 2025년 12월 (총 30년)

수집 항목: 가수명, 곡명, 순위, 가사 전문

전처리 수행:

불용어 제거: 의미 없는 감탄사 제거함 (단, 2회 이상 반복되는 후크는 보존)

형태소 분석: 한글, 영어 라이브러리 활용함

정제: 19세 미만 청취 불가 곡, 연주곡, 해외 곡 제외함

분석 방법

3.1 거시 분석 (구조적 변화)

차트 역동성: 차트에 진입한 신곡의 비율을 통해 고인물 현상을 측정함

정보 밀도: 곡당 평균 단어 수를 분석함

어휘 다양성: 전체 단어 중 고유 단어의 비율로 가사의 단조로움을 측정함

구조적 복잡도: 단어 배치의 불확실성 및 서사 구조의 깊이를 분석함

3.2 거시 분석 (의미적 변화)

군집화: 계층적 군집화를 통해 시대별 가사 트렌드를 그룹핑함

토픽 모델링: 시대별 핵심 담론과 주요 키워드 변화를 추적함

주성분 분석: 가사 트렌드의 연도별 이동 궤적을 시각화함

3.3 미시 분석 (맥락 및 의미론적 변화)

사회적 맥락 분석: IMF, COVID-19 등 주요 사회적 사건과 가사 키워드 간의 상관관계를 분석함

의미론적 변화: 시대별 단어 임베딩 모델 학습 후 벡터 공간 정렬을 통해 동일 단어의 시대별 의미 변화를 추적함

젠더 주체성 분석: 사랑과 이별 관련 키워드의 능동/수동성을 수치화하여 성별 화자의 태도 변화를 측정함

주요 결과

4.1 구조적 분석 결과

차트 고착화: 신곡 진입 비율이 52%에서 17%로 급락하여 특정 곡의 장기 집권이 심화됨

정보량 폭증: 곡당 평균 단어 수가 약 1.6배 증가하여 텍스트 밀도가 상승함

영어 비중 확대: 한국어 중심에서 한영 혼용 및 다국적 텍스트 구조로 변화함

4단계 진화론:

1기(1996~2007): 서정의 시대 (높은 어휘 다양성)

2기(2008~2012): 반복의 시대 (후크송 전성기, 어휘 다양성 최저)

3기(2013~2021): 과잉의 시대 (힙합 대중화, 정보 밀도 폭발)

4기(2022~현재): 경량화의 시대 (듣기 편한 직관적 구조로 회귀)

4.2 의미적 분석 결과 (거시)

시대별 군집:

Cluster 0 (1996~2008): 감성과 서사 중심

Cluster 1 (2009~2019): 아이돌 산업화와 격변기

Cluster 2 (2020~2025): 글로벌 이지 리스닝

트렌드 궤적: 감성적 서사에서 청각적 기능주의로 키워드 중심축이 이동함

4.3 미시 분석 결과 (맥락 및 의미)

사회적 맥락의 투영:

가사는 사건 자체보다 사건으로 인한 라이프스타일 변화를 반영함

IMF: 직접적 언급 부재, 낭만적 도피와 위로의 정서가 우세함

COVID-19: 집, 방, 혼자 등 고립과 주거 관련 키워드가 급증함

주요 키워드의 의미 이동:

여자: 순정, 눈물에서 자신감, 섹시, 언니로 변화함 (수동적 청순에서 주체적 걸크러시로 변화)

침대: 따뜻, 사랑에서 휴대폰, 맥주, 쓸쓸히로 변화함 (로맨틱 공간에서 개인 생활/고독의 공간으로 변화)

첫사랑: 비극, 아픔에서 솜사탕, 축복, 기억으로 변화함 (트라우마에서 아름다운 추억으로 변화)

젠더 주체성 변화:

여성 화자의 가사에서 이별/사랑에 대한 태도가 수동적 기다림에서 능동적 대처로 뚜렷하게 변화함

한계 및 의의

데이터 한계:

멜론 단일 플랫폼 데이터 사용으로 전체 시장 대변에 한계가 존재할 수 있음

월간 차트 기준이므로 주간/일간 차트의 단기 트렌드 반영이 미흡할 수 있음

분석 한계:

19금 곡 및 해외 곡 제외로 인해 힙합 등 특정 장르나 해외 팝 트렌드의 직접적 영향 분석이 제외됨

의의:

30년 시계열 분석을 통해 이지 리스닝과 영어 가사 증가를 단순한 느낌이 아닌 정량적 수치로 증명함

K-POP 가사가 문학적 서사에서 소리의 질감과 미시적 감각을 중시하는 형태로 진화했음을 규명함

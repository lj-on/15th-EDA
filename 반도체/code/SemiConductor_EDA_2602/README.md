# Semiconductor EDA 2602

반도체 EDA 협업 프로젝트를 위한 레포지토리입니다.

## 폴더 구조
- `LithoBench`


각 상위 폴더 아래에는 개인 작업 폴더와 `0_Datasets`가 있습니다.
`0_Datasets`는 용량 이슈로 깃 추적에서 제외됩니다.

## 사용 가이드
1. 레포지토리를 클론합니다.
2. 본인 이름 폴더에서만 작업합니다.
3. 데이터셋은 각자 로컬에서 받아 각 상위 폴더의 `0_Datasets`에만 추가합니다.(git에 업로드/다운로드 안되도록 하였음)
4. 결과물은 본인 폴더 안에 정리합니다.

5. 변경 사항을 회의 전 커밋/푸시합니다.
